# ALMA Core

**Evidence-Based Cognitive Architecture for Long-Term Psychological Understanding**

ALMA Core is the cognitive architecture that powers the ALMA AI research initiative. Rather than generating isolated responses, it incrementally constructs an evidence-based psychological representation that evolves across conversations.

The architecture is designed for longitudinal reasoning, uncertainty management, and cumulative relational understanding.

  ---

## Design Principles

- Evidence before conclusions.
- Multiple competing hypotheses.
- Explicit uncertainty representation.
- Longitudinal memory instead of session memory.
- Reflection before recommendation.
- Psychological understanding rather than prediction.
