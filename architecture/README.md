# ALMA Architecture

The architecture of ALMA is organized into several complementary layers.

Each layer answers a different question about how ALMA works.

---

## System

Defines the overall cognitive architecture.

* System Architecture
* Cognitive Model
* Cognitive Pipeline
* Cognitive State
* Component Interfaces

---

## Components

Defines the major cognitive components responsible for processing psychological understanding.

* Interpretation Engine
* Observation Model
* Pattern Model
* Hypothesis Engine
* Memory System
* Person Model
* Reflection Engine
* Question Engine
* Uncertainty Model

---

## Objects

Defines the core cognitive objects exchanged between components.

* Observation
* Pattern
* Hypothesis

---

## Reasoning

Defines the principles governing psychological reasoning.

* Evidence Model
* Inference Model
* Confidence Model
* Contradiction Model
* Revision Model

---

Together these layers define ALMA's cognitive architecture.

They describe not only what ALMA knows, but how it continuously develops psychological understanding through conversation.
